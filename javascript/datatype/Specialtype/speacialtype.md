# about specialtype  


1. **null** <br>

 `변수에 값이 없음을  의도적으로 알리다`
 
  ```javascript

const amifat = null;
console.log(amifat); // null

 
  ```

   2. **undefined** <br>

      `변수에 값이 정해져있지 않은 상태`
      
      
      ```javascript
      
        let something;

        console.log(something);  // undefined

      ```
